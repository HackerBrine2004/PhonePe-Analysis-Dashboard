# 📱 PhonePe Transaction Analysis Dashboard

An interactive **Power BI dashboard** developed to analyze **PhonePe digital payment transactions** and uncover meaningful business insights. The dashboard provides a comprehensive view of transaction trends, payment categories, state-wise performance, and key business metrics through intuitive and interactive visualizations.

## ✨ Features

- 📊 Analyze total transaction volume and value
- 🗺️ Compare state-wise transaction performance
- 💳 Explore category-wise payment distribution
- 📈 Track monthly and yearly transaction trends
- 📌 Monitor KPIs with interactive cards
- 🎛️ Dynamic filters and drill-down capabilities

## 🛠️ Tech Stack

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)
- Data Modeling
